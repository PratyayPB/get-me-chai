# ‚òï Get Me Chai

**Get Me Chai** is a full-stack crowdfunding platform designed for creators to receive support and donations from their fans. Inspired by "Buy Me A Coffee," it allows users to sign up, create a profile, and start accepting payments in the form of "Chai."

## Authentication Notice
Currently, authentication is available **only via GitHub**.
Other authentication providers are not enabled at the moment.  
Additional login options may be added in future updates.

## üöÄ Features

- **User Authentication**: Secure login using Google OAuth via NextAuth.js.
- **Creator Profiles**: Each user gets a unique profile page where fans can send money.
- **Payment Integration**: Seamless payment processing using the **Razorpay** API.
- **Leaderboard**: Displays the top supporters and recent donations for each creator.
- **Dashboard**: A private dashboard for creators to manage their settings and view earnings.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Theming**: Sleek, modern dark-themed UI built with Tailwind CSS.

### Prerequisites

- Node.js (v18.x or later)
- npm, yarn, or pnpm
- A MongoDB database (Atlas)
- Razorpay API Keys
- Google Cloud Console (for OAuth)
